This project is a 2D isometric MMORPG game client, inspired by classic RPGs such as Runescape classic.

Current Features:
Core Gameplay Systems:
Movement & Pathfinding: Isometric movement and pathfinding logic.
Combat System: Basic entity combat mechanics.
Skills: Dedicated systems for Mining, Woodcutting, and a general Skill System.
Inventory & Items: Management of player items and inventory state.
World & Entities:
Isometric World: Tile-based world with object and definition loaders.
Entities: Support for Players, NPCs, Enemies, and Remote Players (indicating multiplayer).
Animations: System for entity animations.
Networking:
Client-Server Communication: ClientConnection handles network traffic, suggesting a client-server architecture.
UI & Audio:
UI Framework: Custom UI with support for panels, overlays, shops, and quests.
Sound System: Background music (MIDI) and sound effect handling.
Cache Management:
Dynamic Loading: Systems for downloading and loading game assets/cache.
